# Guess Me! (حدس بزن)

A two-player Persian word guessing game built with React, TypeScript, and Vite.

One player enters a secret word, then the other player guesses it letter by letter before running out of attempts.

## Tech Stack

- **React 19** + **TypeScript** + **Vite**
- **Tailwind CSS** - Utility-first styling with RTL support
- **Framer Motion** - Animations and transitions
- **Lucide React** - Icon library

## Features

- Persian (Farsi) RTL interface
- Word category selection
- Real-time letter counting
- Win/loss detection with scoring
- Dark/light theme toggle
- Sound effects (Web Audio API)
- Web Share API integration
- Responsive mobile-first design
- Glassmorphism UI with gradient backgrounds

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
